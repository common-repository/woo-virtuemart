== Description ==
**Virtuemart to woocoommerce**
this plugin imports products, medias, product categories, product attributes from virtuemart to woocommerce

= Features =
* Import all product categories
* Import all product relations
* Import all products
* Import all product medias
* Import all product attributes

= Compatibility =
* Woocommece 3.4.x
* Woocommece 3.5.x
* Wordpress 4.9.x

== Installation ==
1. Upload `virtuemart2woocommerce` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
